Detailed Description

Implementation of the FrameTransformer interface for the conversion of the robot forces into human forces. This class is in charge of converting the forces that are not constant during the physical human-robot interaction. It computes at each timestamp an updating of the transform matrix by means of the human and robot joint configurations.
